This is an unofficial script to see the current value of your fund from Skagenfondene.no.
Currently only supporting Skagen Global, Vekst og Kon-Tiki.

Example output:

```
----------------------------------------
Owner: Name
Investment: 10,000 NOK
return: 1,138 NOK
Total: 11,138 NOK
Return rate: 11.38%
----------------------------------------
```
